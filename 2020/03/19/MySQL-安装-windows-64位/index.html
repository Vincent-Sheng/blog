MySQL 安装(windows 64位)

下载MySql

第一直接下载免费的MySql社区版（免费）
下载链接：https://dev.mysql.com/downloads/

选择MySql Community Server 社区版

选择适合自己的系统

下面选择下载文件

  • msi
    很复杂的全手动安装，都是一步步来的
  • zip
    小白式安装直接解压安装，只需要配置即可

为了方便直接选择winx64的zip安装

1.解压缩下载包

选择你喜欢的路径存放并解压

在F盘新建MySql文件夹，解压缩此文件夹

进入文件夹mysql-8.0.19-winx64

2.配置文件

在使用ZIP安装时，安装好之后默认是没有my.ini配置文件的

解压后的目录并没有的my.ini文件，没关系可以自行创建在安装根目录下添加的my.ini（新建文本文件，将文件类型改为的.ini），写入基本配置：

在mysql-8.0.19-winx64目录下（就是安装根目录（PS：就是直接复制看得到bin文件的路径））

新建my.ini文件
双击记事本打开，写入

初始化MySQL

在安装时，避免出错我们尽量全部使用管理员身份运行CMD，否则在安装时会报错，会导致安装失败的情况

打开cmd.exe程序的时候选择“用管理员身份打开”

cd进入mysql的bin目录

这里先进入F:
再cd F:\MySql\mysql-8.0.19-winx64\bin

输入mysqld --initialize --console （初始化配置，这一步会生成data文件目录等）

这里遇到一个问题

安装mysql时报错: 由于找不到vcruntime140_1.dll,无法继续执行代码

解决方案

解决方法: 这是缺少了动态链接库(.ddl文件), 跟mysql无关, 下载一份即可, 下载地址, 拖到下面, 我下载的是64位版本的压缩包

解压后将vcruntime140_1.dll复制到C:\Windows\System32即可

再执行一次命令

第一次登陆MYSQL时，会提示要求输入初始密码，这是考虑安全因素，命令：mysqld –initialize会随机生成密码。

注意！

[注意] [MY-010454] [服务器]为root @ localhost生成一个临时密码：-AtAgL％V％5sg

其中-AtAgL%V%5sg就是初始密码（不含首位空格）
在没有更改密码前，需要记住这个密码，后续登录需要用到。复制密码先保存起来！！！

tips:如果emm你把它点没了 你只要把datadir配置的那个data的文件删除了，然后重新执行初始化即可
然后输入

安装MySQL服务 + 启动MySQL 服务

安装mysql服务

执行下面的命令：

mysqld –install [服务名]（服务名可以不加默认为mysql）

报错：安装/删除服务被拒绝！

这是权限不够

解决方法：

打开cmd.exe程序的时候选择“用管理员身份打开”

Service successfully installed.返回这个就是成功了

服务安装成功之后通过命令net start mysql启动MySQL的服务

配置环境变量

如果有需要用cmd来运行mysql的话
参考

1.打开我的电脑在我的电脑右键中选择属性，然后单击选择高级系统设置
在系统属性的‘高级’中选择环境变量

2.选择Mysql安装的目录，在Mysql安装的目录中找到bin目录，把此路径复制。

3.在用户变量新建变量添加，变量值为刚复制的路径

4.选择 系统变量 中的中的 path 点击 编辑

建议直接从目录路径拷贝，粘贴上去

5.测试一下

也可以直接cd到mysql的安装路径
F:\MySql\mysql-8.0.19-winx64\bin
输入mysql -u root -p password

新建的MySQL的连接

还记得刚刚让你复制的root @ localhost：后面的初始密码了吗？现在要用到它了复制粘贴上去！

测试一下！

MySQL的是成功安装了也可以登录

现在就是改密码的时候了

可以直接点击刚刚的建的连接就会提示我们输入新密码

也可以在mysql的bin目录下 进行数据库连接 mysql -u root -p

再输入密码 回车

有了mysql> 这个的时候 你就可以去改密码了！

密码修改成功后

exit; 退出mysql
