V2Ray自建翻墙服务器（高速稳定）
2019-1-7 圣圣

  本文只是记录一下博主如何用VPS搭建一个属于自己的V2Ray服务器

VPS

前往Vultr首页 → 注册一个账号

首选国际知名的vultr，速度不错、稳定且性价比高，按小时计费，能够随时开通和删除服务器，新服务器即是新ip，较方便

系统会发一个验证邮件到你的Email，然后激活

登录后你会看见一个支付页面，你可以选择Alipay(支付宝)支付，选择最低的10美元

选择你的服务器地点，一般美西机房、新加坡（自己多测试那个地方的主机比较适合）（PS:貌似日本的都挂了）

官方的测试地址
在本地 ping 测试该节点的延迟和丢包率；
打开测试地址，点击下载测试文件，进一步测试该节点下载速度；确定节点。

服务器选择Ubuntu(系统选新点就好)

选择自己所需要的主机

注意：2.5美元套餐只提供ipv6，如果你用不了ipv6，那么你可以买3.5美元往后的套餐

接下来这一步是开启vps的ipv6，如果你的电脑系统可以用idv6，那么可以勾选此项。大多数用户没有这个需求，但有一些用户可能会用到，所以补充这部分内容。

给自己的主机取个英文名，然后点击Deploy Now

点击下面的图标，，查看主机详细资料

記住IP Address、Username、Password，等等架設主機時會用到

可以先PING一下主机，看主机速度够不够快，不够快就砍掉重建，直到满意为止

请注意！如果连不上，代表主机IP被GFW封锁了，主机也要砍掉重建，一直试到可以连为止

Windows + R 打开运行，输入cmd


下载Xshell并安装，官方地址

（网上很多安装教程，不详细叙述。PS：用其他远程连接软件都操作是大同小异）

新建，然后主机IP输入，确定

先安裝BBR加速器，输入以下代碼(右键粘贴并回车)

接著再输入以下代碼，开始安裝V2Ray (右键粘贴并回车)

如果你之后想卸载V2Ray就输入以下代码

安装好后在输入指令v2ray，回车

按5，更新v2ray和v2ray.fun

在输入一次指令v2ray，设置选项按2更改配置

按1，更改UUID，输入y

按2，更改配置，按2更改主端口，输入40~65535之间的数字（开头不能是0），比如输入:6666

按2再进入更改配置，按3更改加密方式，1~3选一个数字(选一个加密方式)

按2再进入更改配置，按4更改传输方式，1~7选一种传输方式

不同的传输方式会影响连接速度，每个人的网络环境不一样，要自己一一测试看哪一种对自己是最好的

按3，观看服务端讯息，记下这些资料，要连V2RAY用的

(重要!) 按1，服务管理，按1，启动服务

以上是最后一步安装步骤，安裝完后要启动服务，V2RAY才能正常运作。

以后，不管你是更改配置或是重启VPS之后，都一定要再次启动服务，V2Ray才能继续使用。

V2Ray客戶端教程

首先，我们要准备三个东西：

  • 下载安装V2Ray-Core
  • 下载安装V2RayN
  • V2Ray服务器节点账号

以下以Windows系統作演示范例

到V2Ray官网下载核心程式(V2Ray-Core)

有俩个分流下载点

如果你是用Github下载V2Ray，假设你是Windows 64位系统

下载V2RayN客户端

解压缩我们刚才下载的v2ray-windows-64.zip

把刚才下载好的V2RayN.exe拉进v2ray-windows-64.zip解压的文件夹中,双击运行V2RayN.exe

运行之后会弹出警告视图，选择仍要运行（双击右下角的图标可唤出）


（PS：截屏会按不出弹窗，直接手机拍了( • ̀ω•́ )✧ ）

输入V2Ray的资料

如果你当初安装的传输方式是伪装型，这里也要设置（我自己的设置没伪装，所以这部分就选none）

启动HTTP代理

选择适合你的模式，全局模式就是不管国内外网站一律翻墙

开启PAC,并自动配置PAC（PAC模式）就是国内的不翻墙，国外的翻墙

设定完之后，关掉V2RayN ,再重新开启一次（重要！）

再把V2RayN重新打开,就可以翻墙上网了

以上文中演示的主机已删除

这种方式还可以用于手机翻墙，博主下篇文章再写
